# Quick Actions to Add Screenshots

## TL;DR - Do This Now

```bash
# 1. Generate fresh report
cd /Users/satsaugeeth/IdeaProjects/genomic-qa-pgp
mvn clean test

# 2. Open report
open target/cucumber-reports/report.html

# 3. Create screenshot folder
mkdir -p docs/screenshots

# 4. Take 5 screenshots (see below for what to capture)
# Use: Cmd + Shift + 4 on macOS

# 5. Rename and save to docs/screenshots/:
# - 01-report-summary.png (top of report, summary metrics)
# - 02-feature-overview.png (feature list)
# - 03-admin-crud-scenario.png (clicked on Admin CRUD scenario)
# - 04-smoke-test-scenario.png (clicked on Smoke test scenario)
# - 05-known-bug-scenario.png (scrolled to skipped scenario)

# 6. Commit
git add docs/screenshots/
git commit -m "docs: Add test report screenshots"
git push
```

## ✅ Success Criteria

After completing the above:
- [ ] 5 PNG files in `docs/screenshots/`
- [ ] Files are visible on GitHub in the repo
- [ ] README has been updated with screenshot references
- [ ] All screenshots show green/passing tests (except known bug skipped)
